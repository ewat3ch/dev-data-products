The diamonds dataset

The 'Diamonds are Forever' application explores the diamond dataset available with R Package ggplot2. This dataset was built using slidify and hosted on shinyapp.io as Diamonds Are Forever. The application compares the price, cut, carat and clarity of almost 54,000 diamonds.

What carat value of diamond can I get ?

Having chosen either the price, cut or clarity, the application will plot the available diamonds carat within those parameters.

Does carat matter ?

YES! However, in the diamond data set the low carat will be 0.437 and the highest being 5.01.
